0) Conexão com Banco de Dados

1) Cadastro

  - Abrir Postman
  - Selecionar Requisição Postman (POST)
  - Clicar em Body
  - Clicar em raw
  - Selecionar JSON

  - Colar URL
        Testando Localmente: http://localhost:3000/cadatro (Terminal VScode)
        Testando na nuvem : https://maquina-nova.onrender.com/cadastro (Clone com máquina local no Github)

  - Inserir o seguinte corpo 
    {
        "username" : "Samuel Victor",
        "password" : "12345"
    }

  - Clicar em send

(OK)

3) Inserindo Dados
  - Abrir Postman
  - Selecionar Requisição Postman (POST)
  - Clicar em Body
  - Clicar em raw
  - Selecionar JSON

  - Colar URL
        Testando Localmente: http://localhost:3000/inserir_dados_motores (Terminal VScode)
        Testando na nuvem : https://maquina-nova.onrender.com/inserir_dados_motores' (Clone com máquina local no Github)

  - Inserir o seguinte corpo 
{
  "dados_sensores_motores": {
    "Motor01": {
      "Temperatura": 30,
      "Frequencia": 60,
      "Corrente": 3,
      "Vibracao": "ok",
      "Pressao": 1
    },
    "Motor02": {
      "Temperatura": 28,
      "Frequencia": 60,
      "Corrente": 3,
      "Vibracao": "nok",
      "Pressao": 1.5
    }
  }
}

- Clicar em Authorization
- Inserir o Token sem aspas duplas "" fornecido anteriormente nos passos acima para efetuar o post
- Clicar em send

4) Coletando Dados
  - Abrir Postman
  - Selecionar Requisição Postman (GET)

   - Colar URL

        Testando na nuvem : https://maquina-nova.onrender.com/coletando_dados_motores (Clone com máquina local no Github)

        Testando na nuvem : https://maquina-nova.onrender.com/coletando_dados_motores/1 (Clone com máquina local no Github)


  - Clicar em Authorization
  - Inserir o Token sem aspas duplas "" fornecido anteriormente nos passos acima para efetuar o get
  - Clicar em send
